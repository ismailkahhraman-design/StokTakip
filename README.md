# Stok Takip (Vite + React + Tailwind)

## Geliştirme
```bash
npm i
npm run dev
```

## Build
```bash
npm run build
```

## Deploy (Vercel)
- Framework: Vite
- Build: `npm run build`
- Output: `dist`

## Notlar
- Giriş bilgileri: admin / admin
- Veriler tarayıcı localStorage'da saklanır.
- Bulut veriye geçiş için Supabase entegrasyonu planlanmıştır.
